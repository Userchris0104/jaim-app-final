## Usage Tips

1. **Outline icons** (stroke-based) are used for most UI elements
2. **Filled icons** are used for active states and emphasis
3. Use `strokeWidth={1.5}` for navigation icons, `strokeWidth={2}` for action icons
4. Always apply `text-current` color classes to icons for easy theming
